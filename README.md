# FridgeSmart Website — Second Pass

This is the upgraded second-pass Next.js marketing site for FridgeSmart.

## What changed
- More premium homepage layout
- Stronger conversion-focused hero section
- Mobile menu in the header
- Better section hierarchy and messaging
- Improved footer and metadata
- Existing legal pages kept as placeholders

## 1) Add your assets
Place these files inside `public/assets`:

- `fridgesmart-logo.png`
- `owner-avatar.png`
- `chef-lumi-avatar.png`
- `fridgesmart-app-screen.png`

## 2) Install dependencies
```bash
npm install
```

## 3) Run locally
```bash
npm run dev
```

Open:
```bash
http://localhost:3000
```

## 4) Deploy to Vercel
1. Push this folder to GitHub.
2. Import the repo in Vercel.
3. Add your domain in Vercel.
4. Point DNS from GoDaddy to Vercel.

## 5) Main files you will edit most
- `app/page.tsx`
- `components/Header.tsx`
- `components/sections/HeroSection.tsx`
- `components/sections/AboutSection.tsx`
- `components/sections/CtaSection.tsx`
- `app/privacy/page.tsx`
- `app/terms/page.tsx`
